# Concepts de Programmation Courants

Ce chapitre couvre des concepts qui apparaissent dans presque tous les langages de programmation et comment ils fonctionnent en Rust. De nombreux langages de programmation ont beaucoup en commun dans leur cœur. Aucun des concepts présentés dans ce chapitre n’est unique à Rust, mais nous les discuterons dans le contexte de Rust et expliquerons les conventions les entourant.

Plus précisément, vous apprendrez sur les variables, les types de base, les fonctions, les commentaires et le contrôle de flux. Ces fondations seront présentes dans chaque programme Rust, et les apprendre tôt vous donnera un solide socle de départ.

> #### Mots-clés
>
> Le langage Rust a un ensemble de _mots-clés_ qui sont réservés à l’utilisation par le langage uniquement, tout comme dans d'autres langages. Gardez à l'esprit que vous ne pouvez pas utiliser ces mots comme noms de variables ou de fonctions. La plupart des mots-clés ont des significations spéciales, et vous les utiliserez pour effectuer diverses tâches dans vos programmes Rust ; quelques-uns n'ont actuellement aucune fonctionnalité associée, mais ont été réservés pour des fonctionnalités qui pourraient être ajoutées à Rust à l'avenir. Vous pouvez trouver la liste des mots-clés dans [l'Annexe A][appendix_a]<!-- ignore -->.
