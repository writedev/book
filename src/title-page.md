# Le Langage de Programmation Rust

_par Steve Klabnik, Carol Nichols, et Chris Krycho, avec des contributions de la
Communauté Rust_

Cette version du texte suppose que vous utilisez Rust 1.96.0 (publié le 2026-05-28)
ou une version ultérieure avec `edition = "2024"` dans le fichier *Cargo.toml* de tous les projets pour 
les configurer à utiliser les idiomes de l'Édition Rust 2024. Consultez la section [« Installation » du Chapitre 1][install]<!-- ignore --> pour des instructions sur l'installation ou la mise à jour de Rust, et consultez [l'Annexe E][appendix-e]<!-- ignore --> pour des informations sur les éditions.

Le format HTML est disponible en ligne à
[https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/)
et hors ligne avec les installations de Rust réalisées avec `rustup`; exécutez `rustup doc 
--book` pour l'ouvrir.

Plusieurs [traductions] de la communauté sont également disponibles.

Ce texte est disponible en [format broché et ebook chez No Starch Press][nsprust].
