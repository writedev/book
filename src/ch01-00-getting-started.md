# Getting Started

Commençons votre aventure avec Rust ! Il y a beaucoup à apprendre, mais chaque voyage commence quelque part. Dans ce chapitre, nous discuterons de :

- L'installation de Rust sur Linux, macOS et Windows
- L'écriture d'un programme qui affiche `Hello, world!`
- L'utilisation de `cargo`, le gestionnaire de paquets et système de construction de Rust
